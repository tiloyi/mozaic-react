# Accordion

The `Accordion` component is the **React.js** implementation of the **Mozaic Accordion styles**.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/Accordion/).

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source language="javascript" code="import { Accordion } from '@mozaic-ds/react';" />

2. Call the component and give it the desired props, for example:

<Canvas withSource="open">
    <Story story={Controls} />
</Canvas>

Example with check box:

<Canvas withSource="open">
    <Story story={WithCheckBox} />
</Canvas>

Example with toggle:

<Canvas withSource="open">
    <Story story={WithToggle} />
</Canvas>
