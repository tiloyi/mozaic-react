# DataTable

The `DataTable` component is the **React.js** implementation of the **Mozaic DataTable** component.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/DataTable/).

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source language="javascript" code="import { DataTable } from '@mozaic-ds/react';" />
