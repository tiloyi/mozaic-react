# FileUploaderItem

The `FileUploaderItem` component is a partial component to use with FileUploader item in order to display your selected files.

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source
    language="javascript"
    code="import { FileUploaderItem } from '@mozaic-ds/react';"
/>

2. Call the component and give it the desired props, for example:

<Canvas withSource="open">
    <Story story={FileUploaderItemStory} />
</Canvas>

3. You can also give a loading status to your component and give a custom loadingIcon in props, for example:

<Canvas withSource="open">
    <Story story={FileUploaderItemStoryLoading} />
</Canvas>
