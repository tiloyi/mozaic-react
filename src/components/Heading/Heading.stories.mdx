# Heading

The `Heading` component is the **React.js** implementation of the **Mozaic Heading styles**.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Foundations/Typography/HeadingStyles/).

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source
  language="javascript"
  code="import { Heading } from '@mozaic-ds/react';"
/>

2. Call the component and give it the desired props, for example:

<Canvas withSource="open">
  <Story story={Controls} />
</Canvas>
