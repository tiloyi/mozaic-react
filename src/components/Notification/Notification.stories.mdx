# Notification

The `Notification` component is the **React.js** implementation of the **Mozaic Notification** component.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/Notification/).

Для корректной работы нотификаций ваше приложение должно быть обернуто в `NotificationsProvider`.

## Usage

To use the component in your project you must:

1. Import the component where you need it:
