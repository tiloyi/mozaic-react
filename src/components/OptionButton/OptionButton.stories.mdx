# OptionButton

The `OptionButton` component is the **React.js** implementation of the **Mozaic Option button** component.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/OptionGroup/OptionButton/).

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source language="javascript" code="import { OptionButton } from '@mozaic-ds/react';" />

2. Call the component and give it the desired props, for example:

## Do's and Don'ts

  Never use an option button inside a form to choose the civility. Please use
  radio buttons instead.
