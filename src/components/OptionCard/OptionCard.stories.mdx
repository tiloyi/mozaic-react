# OptionCard

The `OptionCard` component is the **React.js** implementation of the **Mozaic Option Card** component.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/OptionGroup/OptionCard/).

## Usage

To use the component in your project you must:

1. Import the component where you need it:
