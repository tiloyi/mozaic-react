# Overlay

Компонент `Overlay` не является самостоятельным в [Mozaic Design System](https://mozaic.adeo.cloud/). Он является общей часть компонентов `Modal` и `Layer`.

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source language="javascript" code="import { Overlay } from 'bigfish-wms--mozaic-react';" />

2. Call the component and give it the desired props, for example:

<Canvas withSource="open">
    <Story story={Controls} />
</Canvas>
