# Pagination

The `Pagination` component is the **React.js** implementation of the **Mozaic Pagination styles**.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/Pagination/).

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source
     language="javascript"
