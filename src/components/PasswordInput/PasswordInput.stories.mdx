# PasswordInput

The `PasswordInput` component is the **React.js** implementation of the **Mozaic PasswordInput** component.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/Form/PasswordInput/).

To use the component in your project you must:
