# QuantitySelector

The `QuantitySelector` component is the **React.js** implementation of the **Mozaic QuantitySelector** component.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/Form/QuantitySelector/).
