# Select

The `Select` component is the **React.js** implementation of the **Mozaic Select** component.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/Form/Select/).

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source language="javascript" code="import { Select } from '@mozaic-ds/react';" />

2. Call the component and give it the desired props, for example:

<Canvas withSource="open">
    <Story story={Controls} />
</Canvas>

or controlled example:

<Canvas withSource="open">
    <Story story={Example} />
</Canvas>

or example with groups:

<Canvas withSource="open">
    <Story story={Groups} />
</Canvas>
