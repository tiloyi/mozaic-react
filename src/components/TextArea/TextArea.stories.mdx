# TextArea

The `TextArea` component is the **React.js** implementation of the **Mozaic TextArea** component.

You can find all the characteristics of this component by going to [the associated documentation](https://mozaic.adeo.cloud/Components/Form/TextArea/).

## Usage

To use the component in your project you must:

1. Import the component where you need it:

<Source
  language="javascript"
  code="import { TextArea } from 'bigfish-wms--mozaic-react';"
/>

2. Call the component and give it the desired props, for example:
