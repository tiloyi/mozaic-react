## Getting started

In order to install `mozaic-react` in your project, you will need to run the following command:

```
npm install --save mozaic-react
```

If you prefer Yarn, use the following command instead:

```
yarn add mozaic-react
```

Once the package installed, import the following `css` file in your project entry point:

```
import 'mozaic-react/lib/index.css';
```
